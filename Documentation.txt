Entry Point Architecture (The Routing)
Root index.html (The Redirector)

Location: /index.html (Project Root)

Technical Role: This is a "dummy" file specifically for GitHub Pages compatibility.

Function: Its only job is to immediately redirect the browser to the real homepage located at public/index.html.

Interaction: It does not load any CSS or JS; it simply handles the initial URL hit.

------------------------------------------------------------------------------------------------------------------------------------------

public/index.html (The Main Landing Page)

Location: /public/index.html

Technical Role: The true "Front Desk" of your application.

Role: Designed to be fast, SEO-friendly, and informative for potential members.

Key Components:

Hero Section: Features a dynamic display powered by slideshow.js.

Public Navigation: A standard header and mobile drawer managed by menu.js.

Information Hub: Sections for Membership plans, Timetables, FAQ, and Contacts.

Authentication Forms: Hidden structures for Login and Registration that are activated by modals.js.

JavaScript Interaction: Loads the full UI suite (faq.js, smooth-scroll.js, slideshow.js, menu.js, and modals.js).
